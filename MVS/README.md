# MULTI-VIEW STEREO (MVS) - Algorithm
## PMVS2 - Patch-based Multi-view Stereo Software 2
![PMVS2](http://www.di.ens.fr/pmvs/)

PMVS é um software desenvolvido por:

*- Yasutaka Furukawa - Estudante de Graduação da Universidade de Illinois em Urbana-Champaign - Postdoc - Universidade de Washington
*- Jean Ponce        - Orientador afilhado    da Universidade de Illinois em Urbana-Champaign, Ecole Normale Supérieure

PMVS software multi-visão estéreo que toma um conjunto de imagens e parâmetros de câmera, e então reconstroi a estrutura 3D de um objeto ou cena visível nas imagens. 
Apenas estruturas rígidas são reconstruidas, em outras palavras, o software automaticamente ignora objetos não-rigidos como pedestres em frente a uma construção. 
A saída do software é um conjunto de pontos orientados ao invés de um modelo poligonal (ou malha) onde coordenadas 3D e a normal de superfície
são estimadas em cada ponto orientado. Esta é a segunda versão do software.
