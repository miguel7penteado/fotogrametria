# STRUCTURE-FROM-MOTION (SfM) - Algorithm
## Bundler
![Bundler](http://www.cs.cornell.edu/~snavely/bundler/)

Bundler é um software desenvolvido por:

*- Noah Snavely - Universidade de Cornell , Cornell Tech, Google Research


Bundler é um sistema de structure-from-motion (SfM) para coleções de imagens não-ordenadas escrito em C e C++. 
Uma versão anterior deste sistema SfM system foi usada no projeto Photo Tourism.
